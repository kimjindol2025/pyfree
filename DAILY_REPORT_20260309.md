### 버그 분석

**버그 1: 산술 연산**
```
print(10 + 5)  →  0 (예상: 15)
print(5 * 2)   →  0 (예상: 10)
```

**근본 원인:**
```
LOAD_CONST [7, 0]      # r7 = 10 ✓
LOAD_CONST [9, 1]      # r9 = 5  ✓
LOAD_CONST [9, 2]      # r9 = 0  ✗ (덮어씀!)
ADD [2, 5, 9]          # r2 = r5(None) + r9(0) = 0
```

**수정 방법:**
1. RegisterAllocator 클래스로 레지스터 추적
2. 할당된 레지스터 중복 금지
3. 함수 호출 시 고정 인자 영역 사용
